Sobre nosotros

No solo una página web si no un punto de reunión

Eventurate es una plataforma web creada por un grupo de estudiantes de la Universidad Peruana de Ciencias Aplicadas (UPC), los cuales buscan sosegar la necesidad de muchas personas al buscar u obtener información de un evento.

Programación web

Esta plataforma fue creada como un presente trabajo para el curso de programación web de la Universidad Peruana de Ciencias Aplicadas

Desarrollo

Eventurate está desarrollado con la tecnología de Springboot y sus diferentes herramientas que la complementan. Asimismo, se utilizó para los estilos las facilidades que brinda Bootstrap.

Planeación

El desarrollo fue realizado con apoyo de plataformas como github para el desarrollo colaborativo y trello para la planeación del proyecto

Propósito

Buscamos facilitar y reducir la necesidad de información de eventos locales o virtuales, con el propósito de de aumentar ventas para los negocios y que las personas encuentren diferentes formas de recrearse
